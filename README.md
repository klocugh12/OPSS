This repository contains solutions for problems from now defunct opss.safo.biz website.
Block comment for each problem provides descriptions of task, input and output data.
Each block comment contains subjective difficulty rating of task out of 5. Problemset refers to number of contest tasks appeared in, using chronological order. Problemset 0 means problems appeared outside of contests. Only first problem that appeared in different problemset than previous ones is marked with annotation.
Routines are not reused to mimic original character of solutions as one-and-done applications.
