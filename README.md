This repository contains solutions for problems from now defunct opss.safo.biz website.

Block comment for each problem provides descriptions of task, input and output data.
Each block comment also contains subjective difficulty rating of task out of 5, and number of problemset it belongs to.

Problemset refers to number of contest tasks appeared in, using chronological order, or 0, if it appeared outside of any contest. If not annotated, a problem appeared in same problemset as last problem that had this annotation.

Routines are not reused to mimic original character of solutions as one-and-done applications.
